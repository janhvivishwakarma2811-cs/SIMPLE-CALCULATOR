🧮 SIMPLE CALCULATOR(PYTHON)
📌 Description

This is a simple command-line calculator written in Python. It allows users to perform basic arithmetic operations like addition and subtraction.

⚙️ Features
Addition of two numbers
Subtraction of two numbers
User-friendly menu
Continuous execution until user exits
🚀 How to Run
Install Python on your system

Run the program using:

python calculator.py
🧠 Concepts Used
Functions
Loops (while loop)
Conditional statements (if-elif-else)
User input handling
📷 Example
Welcome to Calculator

Choose one operation:
1. Add
2. Subtract
3. Exit
